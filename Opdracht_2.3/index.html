Onderzoek naar HTML, CSS en Browsers

Waarom noem je de eerste HTML-pagina; index.html?

De naam index.html wordt gebruikt omdat het de standaard startpagina is die de server laadt wanneer er geen pagina is opgegeven.

Met welke HTML versie werken we nu?

We werken met HTML5, de nieuwste versie van HTML.

Met welke CSS versie werken we nu?

De meest recente versie van CSS is CSS3, dat nieuwe lay-out mogelijkheden en stijlen heeft toegevoegd.

Hoe kan het dat een website er net iets anders uitziet in een andere browser?

Verschillende browsers gebruiken verschillende code, wat kan leiden tot kleine verschillen in de uiterlijk van websites.

Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

De populairste browser is Google Chrome. Het RAM gebruik hangt af van de website, maar Chrome heeft een hoog RAM gebruik per tabblad, vergeleken andere browsers net als Brave die ik altijd gebruik.
